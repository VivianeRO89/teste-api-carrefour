# Teste API Carrefour

Projeto de automação de testes de API utilizando Bruno e GitHub Actions.

## Cenários Automatizados

- Criar usuário
- Fazer login
- Listar usuários
- Buscar usuário por ID
- Atualizar usuário
- Excluir usuário
- Validar usuário não encontrado

## Ferramentas Utilizadas

- Bruno
- GitHub
- GitHub Actions

## Execução Local

```bash
bru run .
```

## Integração Contínua

Os testes são executados automaticamente via GitHub Actions a cada alteração enviada para a branch main.
